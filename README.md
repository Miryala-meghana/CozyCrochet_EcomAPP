📌 Project Description:

Cozy Crochet is a simple E-Commerce web application designed to showcase and browse crochet products. The application allows users to view products, search for items, filter by category, and sort products by price for a better shopping experience.

The frontend of the application is built using React with Vite, styled with Bootstrap to provide a responsive and modern user interface. The backend is developed using JavaScript, which handles product and category data through APIs that the frontend consumes.

The application demonstrates key e-commerce functionalities, including product listing, category filtering, search capability, price sorting, and theme switching (Dark Mode / Light Mode). It is designed to be simple, responsive, and user-friendly.

This project highlights the integration of frontend frameworks with backend APIs, showcasing full-stack development concepts and modern web development practices.

Features

✔ View all crochet products

✔ Search products by name

✔ Filter products by category

✔ Sort products by price (Low → High / High → Low)

✔ Responsive product cards using Bootstrap

✔ Dark Mode / Light Mode toggle

✔ Clean and modern UI

⚙️ Installation & Setup:

1️) Clone the repository
    git clone https://github.com/your-username/cozy-crochet.git
2️) Navigate to the project
    cd cozy-crochet
3️) Install frontend dependencies
     npm install
4️) Run the frontend application
    npm run dev

   The application will run on:

   http://localhost:5173
   
5️) Run the backend server

   Make sure your backend API is running on:

   http://localhost:8080

Available endpoints:

GET /api/products
GET /api/categories
